# Getting Started

## System Requirements
* A PC running [X-Plane 11.10](http://x-plane.com) or newer with suitable graphics hardware
* Windows 7, 8.1, 10
* [Microsoft .NET Framework](https://dotnet.microsoft.com/download/dotnet-framework) 4.7.2 or higher
* **CSL Package** see the [installation](#) section for more details.
	* [Bluebell OBJ8 CSL Package](https://forums.x-plane.org/index.php?/files/file/37041-bluebell-obj8-csl-packages/). It is highly recommended you install the **Vertical Offsets** to prevent landsharks (aircraft sunken into the ground).

If in doubt about X-Plane hardware requirements, please check the official [X-Plane 11 System Requirements](http://www.x-plane.com/kb/x-plane-11-system-requirements/).
